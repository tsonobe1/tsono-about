# 開発ログ

Nuxt Content を入れて、記事一覧と詳細ページを作りました。
ページの構造をシンプルにして、どこを触れば見た目が変わるのか分かりやすくしています。

## やったこと

1. `content/` に Markdown を追加
2. `/` に一覧ページを作成
3. `[...]` ルートで詳細ページを作成
4. ベースのスタイルを CSS で定義

## 感想

シンプルな構成なので、今後は記事を増やすだけで更新できます。
デザインを変えるときも、`assets/css/tailwind.css` を触れば十分です。

## 次にやりたいこと

- 記事に日付を表示する
- タグで絞り込みできるようにする
- 目次を自動生成する
